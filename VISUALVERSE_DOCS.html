VisualVerse

VisualVerse is an intelligent, dual-mode text-visualization system that automatically transforms any input text into either a Comic Strip (for narrative/story text) or an Interactive Mind Map (for informational/conceptual text) using a full custom-trained NLP pipeline.

Problem Statement

📄

Long Text = Low Comprehension

Dense paragraphs are hard to parse and retain by learners.

🔍

Abstract Concepts

Conceptual knowledge is difficult to visualize without external tools.

🛠️

No Unified Tool

No existing system converts text to both comics AND mind-maps automatically.

🌏

Multilingual Gap

Most NLP tools lack support for regional Indian languages like Hindi and Tamil.

Key Features

🎭

Dual-Mode Output

Single system produces Comics for narrative text and Mind Maps for informational text.

🤖

Auto Classification

Random Forest classifier automatically detects text type — no manual selection needed.

📖

Story Generator

Keywords → 500-800 word stories using fine-tuned distilGPT-2 across EN/HI/TA.

🖼️

AI Image Generation

SD 1.5 + LCM LoRA generates comic panel images from SVO-structured prompts.

def preprocess(text, language="en"):
    nlp = spacy.load("en_core_web_sm")
    doc = nlp(text)
    return {
        "tokens":       [t.text for t in doc],

        "entities":     [(e.text, e.label_) for e in doc.ents],
        "dependencies": [(t.text, t.dep_, t.head.text) for t in doc],
        "language":     detected_lang
    }

Topic Modeling

Dual approach: LDA (Latent Dirichlet Allocation) for probabilistic topic discovery + semantic clustering for concept grouping.

📊

LDA

Treats the document as a mixture of topics; each topic is a distribution over vocabulary words.

🔗

Semantic Clustering

Groups related keyphrases by word embedding similarity into domain categories.

🗂️

BERTopic

Transformer-based clustering using sentence embeddings for fine-grained topic discovery.

Output: Main topic label, category groupings (e.g., Frontend / Backend / Database), and topic-word distributions used to build the mind map hierarchy.

LCM LoRA (Latent Consistency Model) enables 4-step image generation on CUDA, making it fast enough for real-time panel synthesis on a 4GB VRAM GPU.

Mind Map Generator

Builds a hierarchical NetworkX DiGraph from extracted keyphrases, topics and relations, rendered interactively on the frontend using D3.js.

🎯

Level 0 — Center

Main topic node derived from LDA / BERTopic output.

🗂️

Level 1 — Categories

Topic clusters (e.g. Frontend, Backend, Database) fan out from center.

🔤

Level 2 — Details

Individual keyphrases attached to their parent category.

graph = networkx.DiGraph()
graph.add_node("center", label=main_topic, level=0)
for cat in categories:
    graph.add_edge("center", cat_id)      # level 1
for kp in keyphrases:
    graph.add_edge(parent_cat, kp_id)   # level 2

The frontend D3MindMap.tsx component renders nodes with force-directed layout, supports zoom/pan, and color-codes by level.

Story Generator

A dedicated 4th mode: users supply keywords and the system generates a 500-800 word story using either Llama 3.2 3B (production) or fine-tuned distilGPT-2 (fallback), then optionally feeds the result into the comic pipeline.

Conclusion

VisualVerse successfully demonstrates a complete, production-grade NLP pipeline — from raw text preprocessing through custom-trained classification, keyphrase extraction, topic modeling and relation extraction — to multi-modal visual output generation via comic panels and interactive mind maps. The system supports English, Hindi and Tamil, runs on consumer 4GB VRAM GPUs, and is cloud-deployed via Docker on Render.
